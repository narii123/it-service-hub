# IT Service Hub

A web-based platform connecting Nigerian IT professionals with international clients for IT services.

---

## 🚀 Features

- Role-based authentication (Admin, Client, Professional)
- Service request system
- Job posting and applications
- Project creation and tracking
- Deliverables submission & review
- Platform fee calculation
- Dashboard for each user role

---

## 🧱 Tech Stack

**Frontend**
- React
- CSS

**Backend**
- Node.js
- Express

**Database**
- MongoDB

---

## 🔐 Security

Sensitive data is stored in a `.env` file and excluded from version control using `.gitignore`.

---

## 📊 System Workflow

1. Client creates service request  
2. Admin reviews and approves  
3. Admin converts to job post  
4. Professional applies  
5. Admin selects professional  
6. Project is created  
7. Professional submits deliverables  
8. Admin reviews and completes project
